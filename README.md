# Tarik Tambang

Tarik tambang adalah sebuah permainan tanya jawab antara dua tim. Setiap tim akan diberikan pertanyaan, dan tim yang menjawab dengan benar akan mendapatkan poin. Tim yang memiliki poin terbanyak setelah beberapa putaran akan menjadi pemenang.

## Running

### Windows
```
npm run install
npm run build
New-Item resources/database.sqlite
php artisan migrate
php artisan serve
```

### Linux
```
npm run install
npm run build
touch resources/database.sqlite
php artisan migrate
php artisan serve
```
